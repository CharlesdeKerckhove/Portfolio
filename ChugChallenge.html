Chug Challenge

This mobile application is a drinking app targetted towards students and young adults. Its simple design and user-friendly controls allow users to navigate throughout the three games included in this application. The bulk of this application is a randomized element stored within an array being called and displayed for users to read and react to. The rules of Chug Challenge are displayed and each round creates two rules challengers must follow whilst also adhering to the instructions displayed on the device.

This application was built through PhoneGap and therefore used HTML, CSS and JavaScript to function. Plans for graphics, a database and PHP to be added by myself have already begun meaning the final release should occur in September.
